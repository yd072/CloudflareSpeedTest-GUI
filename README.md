# 🚀 CF-SpeedHub (CloudflareSpeedTest 图形化工具)


[![Core-Version](https://img.shields.io/badge/Core--Version-v2.3.5-green.svg)](https://github.com/XIU2/CloudflareSpeedTest/releases)

`CF-SpeedHub` 是专为 XIU2 大神的经典开源项目 `CloudflareSpeedTest` 打造的**图形化外壳 (GUI Wrapper)**

让你彻底告别繁琐的命令行输入，点点鼠标，即可压榨出宽带网络下对 Cloudflare 节点的最强性能。

📂 CloudflareSpeedTest-GUI/
├── 📄 gui_wrapper.exe      (图形化客户端)
├── 📄 CloudflareST.exe    (XIU2 官方测速核心引擎)
├── 📄 ip.txt              (官方 IPv4 节点段数据)
└── 📄 ipv6.txt            (官方 IPv6 节点段数据)
